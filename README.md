# 🚛 Trans Shirley - Site Institucional

Site institucional moderno e responsivo para a empresa de transporte e logística Trans Shirley. Desenvolvido com HTML5, Tailwind CSS, Alpine.js e ícones Lucide, o projeto apresenta a história, serviços, cobertura nacional e canais de contato da empresa.

## 📋 Visão Geral

A Trans Shirley atua há 15 anos no ramo de transporte nacional, oferecendo soluções completas em logística com frota renovada e profissionais qualificados. O site foi projetado para transmitir confiança, modernidade e profissionalismo, com foco em conversão (orçamentos e contato).

**Funcionalidades principais:**
- Layout totalmente responsivo (mobile-first)
- Menu com navegação sticky e mobile toggle
- Páginas: Home, Sobre, Serviços, Cobertura, Contato
- Formulário de contato integrado (via Formspree)
- Mapas incorporados (Google Maps) para as duas unidades
- Animações suaves e ícones vetoriais

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| [Tailwind CSS](https://tailwindcss.com/) | Framework CSS utilitário para estilização rápida e consistente |
| [Alpine.js](https://alpinejs.dev/) | Framework JavaScript leve para interatividade (menu mobile, formulário) |
| [Lucide](https://lucide.dev/) | Biblioteca de ícones vetoriais (substitui FontAwesome) |
| [Formspree](https://formspree.io/) | Serviço de backend para envio de formulários (opcional) |
| Google Maps Embed API | Mapas estáticos incorporados (substitua pelos IDs corretos) |
